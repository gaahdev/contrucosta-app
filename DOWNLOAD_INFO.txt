COMO USAR:
═══════════════════════════════════════════════════════════════

WINDOWS:
  1. Fazer download de Contrucosta.App.exe
  2. Clique duplo no arquivo
  3. Se aparecer aviso: "Executar mesmo assim"
  4. Aplicação abrirá automaticamente

  2. Executar: adb install app-debug.apk
  3. Abrir "Contrucosta App" no dispositivo

  2. Clicar no arquivo no dispositivo

WEB (LOCAL):

  2. Acessa: http://localhost:3000
